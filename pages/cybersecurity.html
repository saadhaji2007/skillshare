In this course, you’ll learn the building blocks of the web: HTML, CSS, and JavaScript. We’ll start with structure and styling, and gradually move to dynamic content and responsiveness.

You’ll build real projects such as personal portfolios, landing pages, and forms. By the end of the course, you’ll be able to create your own fully responsive and interactive website.
